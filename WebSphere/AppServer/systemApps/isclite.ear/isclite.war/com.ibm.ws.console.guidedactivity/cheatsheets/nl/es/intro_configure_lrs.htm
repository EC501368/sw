Además de las aplicaciones J2EE transaccionales a las que WebSphere Application Server da soporte, WebSphere Extended Deployment Compute Grid da soporte a trabajos nativos, de cálculo intensivo y por lotes transaccionales. Este soporte permite ampliar la infraestructura de WebSphere Application Server para que aloje aplicaciones cuyo modelo de proceso no corresponde al paradigma tradicional de petición/respuesta de J2EE. WebSphere Extended Deployment permite desplegar estos tipos de aplicaciones junto con aplicaciones J2EE transaccionales y utiliza las políticas de servicio definidas administrativamente para asignar recursos de forma dinámica entre todo el conjunto de aplicaciones.

Supuestos

  • Están instalados los niveles correctos de WebSphere Extended Deployment.
