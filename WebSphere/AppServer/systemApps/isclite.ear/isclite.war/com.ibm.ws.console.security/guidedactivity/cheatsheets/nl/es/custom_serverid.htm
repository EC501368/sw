Consejos y sugerencias

Consideraciones especiales sobre los registros de usuario personalizados

Compruebe que la implementación del registro de usuarios personalizado no depende de ningún componente de WebSphere Application Server como orígenes de datos, enterprise beans, etc. No debe existir esta dependencia porque la seguridad se inicializa y habilita antes que la mayoría de componentes de WebSphere Application Server durante el arranque. Si las implementaciones anteriores utilizaban estos componentes, haga los cambios necesarios para eliminar la dependencia.
