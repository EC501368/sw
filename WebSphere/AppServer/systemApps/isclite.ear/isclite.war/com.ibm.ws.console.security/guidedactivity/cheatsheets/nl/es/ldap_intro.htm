Suposiciones

Esta hoja de apuntes describe los valores básicos que se necesitan para que WebSphere Application Server autentique usuarios con un servidor de directorios LDAP (Lightweight Directory Access Protocol). Se supone que el servidor LDAP existe y que se ha cumplimentado con usuarios y grupos. Para obtener más información sobre esta tarea, consulte los siguientes artículos del Information Center:
  • Configuración de los registros de usuarios de LDAP (Lightweight Directory Access Protocol)
  • Utilización de servidores de directorios específicos como servidor LDAP
  • Autenticación de usuarios con un registro de usuarios LDAP
  • Autenticación de usuarios con un registro de usuarios LDAP

Antes de empezar

  • Configure un servidor de directorios compatible con LDAP o compruebe que está disponible un servidor de directorios para almacenar información de usuarios y grupos tanto para las aplicaciones como para la administración de WebSphere Application Server.
  • Cree los usuarios y grupos que sean necesarios dentro del directorio. Nota: WebSphere Application Server no proporciona gestión directa de la información de usuarios y grupos dentro del propio directorio.
