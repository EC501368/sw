Suposiciones

Esta hoja de apuntes describe los valores básicos que se necesitan para que WebSphere Application Server autentique usuarios con ID de servidor de sistema operativo local. Se supone que ya existe un ID de servidor en el registro de usuarios y que tiene asignadas las autoridades correctas.

Antes de empezar

Consulte los siguientes artículos del Information Center para ver las instrucciones de instalación:
  • Registros de usuarios del sistema operativo local
  • Configuración de registros de usuarios del sistema operativo local
