Consejos y sugerencias

Consideraciones especiales sobre los ID de servidor

El ID de servidor puede ser cualquier usuario válido del registro de usuarios y no es necesario que sea el mismo que el identificador del sistema operativo bajo el que se ejecuta el proceso del servidor de aplicaciones. No es necesario que este ID sea root en plataformas UNIX y no es necesario que sea un administrador en plataformas Windows. El ID de servidor utilizado por el servidor de aplicaciones tiene su propia identidad para los propósitos siguientes:
  1. El servidor de aplicaciones correlaciona implícitamente el ID de servidor con el rol de administrador para que el servidor de aplicaciones pueda acceder a los datos de configuración importantes e invocar acciones privilegiadas en tiempo de ejecución, que requieren el rol de administrador.
  2. El servidor de aplicaciones utiliza el ID de servidor y la contraseña para autenticar a otros WebSphere Application Servers en un entorno distribuido.

ID de servidor frente a identidad de proceso

Cada proceso del servidor de aplicaciones también tiene una identidad de proceso que puede ser diferente del ID de servidor aquí especificado. La identidad de proceso es el ID de usuario bajo el que se ejecuta el proceso del servidor de aplicaciones.

Por ejemplo, un proceso de servidor necesita ejecutarse bajo la cuenta root para acceder a interfaces de programación privilegiadas del sistema operativo cuando utilice un registro de usuarios de sistema operativo local en plataformas UNIX.

Para obtener más información, consulte el artículo del Information Center de WebSphere Application Server titulado Configuración de registros de usuarios del sistema operativo local.

Recomendaciones sobre el uso administrativo del ID de servidor

No necesita correlacionar explícitamente el ID de servidor con el rol administrativo en la consola administrativa. Es aconsejable no utilizar el ID de servidor para temas administrativos cotidianos. Sin embargo, si es muy aconsejable correlacionar al menos otro ID de usuario con el rol administrativo.
