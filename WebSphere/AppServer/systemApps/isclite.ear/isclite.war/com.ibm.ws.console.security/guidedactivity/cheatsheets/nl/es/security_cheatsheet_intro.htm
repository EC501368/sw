Suposiciones

Esta hoja de apuntes sólo describe el conjunto de pasos más básicos para habilitar la seguridad global para que el acceso a la consola administrativa esté restringido a usuarios autorizados. Para obtener más información, consulte las siguientes fuentes en el Information Center:
  • Planificación de la seguridad del entorno
  • Protección del entorno administrativo

Antes de empezar

  • Compruebe que está configurado un registro de usuarios soportado o configure un nuevo registro de usuarios. Para obtener más información, consulte el artículo del Information Center titulado Registros de usuarios.
  • Defina los usuarios y grupos en el registro de usuarios que necesiten acceso administrativo utilizando las herramientas de gestión proporcionadas por el proveedor.

Información necesaria

  • Tipo de registro de usuarios que tiene previsto utilizar, incluida la información para conectar con el registro de usuarios, si es necesario.
  • Usuarios que tiene previsto que administren el sistema.

Cuando finalice

Después de completar la hoja de apuntes, deberá utilizar un ID de usuario y contraseña para iniciar la sesión en la consola administrativa porque habrá habilitado la seguridad.
