Suposiciones

Se supone que el servidor Web y el servidor de aplicaciones están en nodos diferentes. Se suele hacer referencia a este entorno como servidor Web "remoto" porque no es local para el servidor de aplicaciones. Para obtener más información, consulte las siguientes fuentes en el Information Center:
  • Configuración de un servidor Web y un servidor de aplicaciones en máquinas distintas (remotas para Distribuido y zOS)
  • Configuración de un servidor Web y un servidor de aplicaciones en máquinas distintas (remotas para OS/400)
  • IBM Education Assistant

Antes de empezar

  • Compruebe que el servidor Web está soportado por WebSphere Application Server. Consulte la página Web de soporte de WebSphere Application Server para obtener la información actualizada sobre software y hardware soportados.
  • Compruebe que el software del servidor Web está instalado correctamente y que el servidor puede aceptar peticiones.

Cuando finalice

Después de completar la hoja de apuntes habrá definido un servidor Web en la configuración de WebSphere Application Server que puede direccionar peticiones de contenido dinámico hacia el servidor de aplicaciones adecuado.
