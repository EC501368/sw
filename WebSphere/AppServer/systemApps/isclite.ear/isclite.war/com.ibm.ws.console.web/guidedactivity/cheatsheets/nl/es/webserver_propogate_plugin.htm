Consejos y sugerencias

Cortafuegos

Compruebe que tiene abiertos los puertos correctos en los cortafuegos que se encuentran entre las máquinas servidor de aplicaciones y las máquinas que ejecutan los servidores Web. El cortafuegos debe tener el mismo puerto administrativo abierto para que el archivo del plug-in pueda pasar. Consulte al administrador de seguridad de la red para comprobar que el puerto está abierto.

Intervalos de sondeo

En el caso de un plug-in HTTP, la duración del retardo que se produce entre consultas de nuevas versiones del archivo de configuración del plug-in viene determinado por el atributo Intervalo de renovación del elemento de configuración en el archivo plugin-cfg.xml. El plug-in sondea el disco con este intervalo para determinar si ha cambiado la configuración. El intervalo por omisión es de 60 segundos.
