Consejos y sugerencias

  • El nombre de nodo es simplemente un nombre lógico para un grupo de servidores. Puede utilizar el nombre que desee. El nombre de nodo no tiene que ser el nombre de host.
  • Los agentes de nodo son agentes administrativos que representan un nodo en el sistema y gestionan los servidores de dicho nodo. Los agentes de nodo supervisan los servidores de aplicaciones en un host y direccionan las peticiones administrativas hacia los servidores. Cuando se añade un nodo en la célula se crea automáticamente un agente de nodo.
