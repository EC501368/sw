Propiedades personalizadas del clúster dinámico

Esta página sirve para crear o suprimir propiedades personalizadas. Para que las propiedades personalizadas funcionen correctamente, las parejas de nombre y valor han de tener alguna relevancia. Las propiedades personalizadas establecen propiedades de configuración del sistema interno, de forma que el establecerlas requiere un conocimiento significativo del producto.

Para ver esta página de la consola administrativa, pulse Servidores > Clústeres > Clústeres dinámicos > nombre_clúster_dinámico > Propiedades personalizadas.

Es necesario poseer privilegios de configurador para crear o suprimir clústeres dinámicos. Consulte la sección de referencia del centro de información para obtener información sobre otros roles administrativos y privilegios para los clústeres dinámicos.

Nombre

Especifica un nombre exclusivo para la propiedad personalizada.

Cada nombre de propiedad debe ser exclusivo. Si se utiliza el mismo nombre para varias propiedades, se utiliza el valor especificado para la primera propiedad con dicho nombre. Los nombres de propiedades no deben empezar por was, porque este prefijo está reservado para las propiedades predefinidas en WebSphere Application Server.

Valor

Especifica el valor que está emparejado con el nombre especificado.

Descripción

Proporciona información acerca del nombre y el valor de la pareja.
