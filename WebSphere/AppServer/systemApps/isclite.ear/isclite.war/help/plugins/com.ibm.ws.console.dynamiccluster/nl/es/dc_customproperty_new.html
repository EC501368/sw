Propiedades personalizadas del clúster dinámico

Utilice esta página para crear propiedades para el miembro de un clúster dinámico. Para ver esta página de la consola administrativa, pulse Servidores > Clústeres dinámicos > Seleccione un clúster existente > Propiedades adicionales (Propiedades adicionales) > Nuevo.

Para que las propiedades personalizadas funcionen adecuadamente, es necesario que las parejas nombre/valor sean relevantes. Crear propiedades personalizadas requiere un conocimiento significativo del producto. En caso de crear una propiedad personalizada que no tenga valor real en el código, no ocurrirá nada.

Los atributos siguientes definen una propiedad personalizada:
NombreEl campo de nombre de una propiedad personalizada es un campo necesario que debe contener un valor exclusivo.
ValorLas entradas de valores válidos de este campo se pasan como una serie.
DescripciónEl campo de descripción es opcional, pero se recomienda en entornos con varios administradores.
