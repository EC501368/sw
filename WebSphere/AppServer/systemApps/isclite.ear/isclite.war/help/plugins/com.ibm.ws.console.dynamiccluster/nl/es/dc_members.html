Colección de miembros de clústeres dinámicos

Utilice esta página para ver, iniciar y detener los miembros de un clúster dinámico.

Para ver esta página de la consola administrativa, pulse Servidores > Clústeres > Clústeres dinámicos > nombre_clúster_dinámico > Miembros del clúster dinámico.

Es necesario poseer privilegios de operador para ver, iniciar y detener miembros de un clúster dinámico.

Para iniciar o detener un miembro, selecciónelo y pulse Iniciar o Detener.

Si elige definir de forma manual los miembros del clúster dinámico, puede añadirlos o eliminarlos. Pulse Añadir o Eliminar.

Nombre de miembro

Especifica el nombre del miembro del clúster dinámico.

El sistema proporciona de forma automática un nombre para un miembro del clúster dinámico. El sistema utiliza la siguiente nomenclatura: <nombre_clúster_dinámico>_<nombre_nodo>.

Nodo

Especifica el nodo en que está en ejecución el miembro del clúster dinámico. Al crear un clúster dinámico, se crea un miembro del clúster dinámico de forma automática en un nodo de forma que se ajuste a la política de pertenencia creada.

Versión

Especifica la versión del producto WebSphere Application Server o WebSphere Extended Deployment que se ejecuta el miembro de clúster.

Peso configurado

Especifica el peso actualmente configurado para el miembro del clúster. El peso determina la cantidad de trabajo que se dirige al servidor de aplicaciones. Si el valor del peso del servidor es mayor que los valores de peso asignados a otros servidores en el clúster, el servidor recibe un parte mayor de la carga de trabajo del clúster.

Para cambiar el peso configurado de un miembro de clúster, especifique un nuevo peso y, a continuación, pulse Actualizar, o pulse en el nombre del miembro del clúster.

Importante: Si se habilita, el gestor de carga de trabajo dinámica (DWLM) altera temporalmente todos los valores que se entren. DWLM está habilitado por omisión. Para inhabilitar DWLM, pulse Servidores > Clústeres > Clústeres dinámicos > nombre_clúster_dinámico > Gestión de carga de trabajo dinámica (DWLM)

Peso de tiempo de ejecución

Especifica la carga de trabajo proporcional que actualmente se dirige al miembro del clúster respecto al peso configurado para dicho miembro de clúster. Si desea cambiar la proporción, especifique un nuevo peso y pulse Actualizar.

Importante: Si se habilita, el gestor de carga de trabajo dinámica (DWLM) altera temporalmente todos los valores que se entren. DWLM está habilitado por omisión. Para inhabilitar DWLM, pulse Servidores > Clústeres > Clústeres dinámicos > nombre_clúster_dinámico > Gestión de carga de trabajo dinámica (DWLM)

Estado

Especifica el estado actual del miembro del clúster. Mantenga pulsado el ratón sobre el gráfico para determinar el significado del gráfico de estado.

Icono Estado Descripción
Iniciado El servidor está en ejecución.
Parcialmente iniciado El servidor está en el proceso de cambiar de un estado de Detenido a un estado de Iniciado. Se está comenzando a ejecutar el servidor pero todavía no lo está haciendo en su totalidad.
Detenido El servidor no está en ejecución.
Parada parcial El servidor está en el proceso de cambiar de un estado de Iniciado a un estado de Detenido.
No disponible No se puede determinar el estado.
No aplicable El servidor no proporciona información sobre si está en ejecución.
