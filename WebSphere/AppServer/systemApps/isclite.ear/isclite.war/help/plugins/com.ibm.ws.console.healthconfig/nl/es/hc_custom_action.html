Valores de acciones personalizadas

Permite especificar detalles para las acciones personalizada de Java y las que no lo son.

Para ver esta página de la consola administrativa, pulse Políticas operativas > Acción personalizada > nombre_acción_personalizada.

Nombre

Especifica un nombre para la acción personalizada.

Ubicación del archivo ejecutable de Java

Especifica la ubicación del archivo ejecutable de Java, por ejemplo ${JAVA_HOME}. Se puede especificar una variable o una vía de acceso. Este campo solo es para acciones Java.

Tipo de destino ejecutable

Especifica el tipo de archivo que se utiliza para el archivo ejecutable de Java. Este campo solo es para acciones Java.

El destino ejecutable puede ser de uno de los siguientes tipos:

  • EXECUTABLE_JAR
  • JAVA_CLASS

Archivo ejecutable

Especifica el archivo que se ejecuta cuando se incumple la condición de salud.

Argumentos ejecutables

Especifica los argumentos que se pasan al archivo ejecutable.

Por ejemplo, se podría pasar el siguiente argumento:

-server ${WAS_SERVER_NAME}

Sistemas operativos soportados

Especifica los sistemas operativos a los que se da soporte con el archivo ejecutable de acción personalizada.

  • Windows
  • Linux
  • AIX®
  • HP-UX
  • Solaris
  • OS/400®
  • OS/390®

Directorio de trabajo

Especifica la ubicación en el sistema de archivos desde donde se invoca el mandato.
