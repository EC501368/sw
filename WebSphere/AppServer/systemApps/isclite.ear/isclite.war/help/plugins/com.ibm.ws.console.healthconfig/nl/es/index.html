Políticas de salud

Vaya a los temas de ayuda de esta sección.

  • Creación de una política de salud: definir las propiedades generales de la política de salud
    Utilice esta página para crear políticas de salud, utilizadas para realizar una gran variedad de cálculos basados en salud sobre los clústeres, clústeres dinámicos e instancias de servidores de aplicaciones que se ejecutan en los nodos.
  • Creación de una política de salud: definir las propiedades de la condición de salud
    Utilice esta página para definir el comportamiento específico de la política de salud.
  • Creación de una política de salud: especificar los miembros a supervisar
    Utilice esta página para asignar propiedades de pertenencia a políticas de salud.
  • Colección de acciones personalizadas
    De forma predeterminada, se puede definir una política de salud para ejecutar cualquiera de las acciones de salud predefinidas, tal como reiniciar un servidor, cuando se ha incumplido una política de salud. Configure una acción personalizada cuando desee que el entorno realice una acción distinta a la especificada por omisión.
  • Valores de acciones personalizadas
    Permite especificar detalles para las acciones personalizada de Java y las que no lo son.
  • Generador de subexpresiones de condición de salud personalizada
    El generador de subexpresiones de condición de salud permite definir una condición de salud personalizada para su política de salud. El programa de utilidad de generación de subexpresiones sirve para generar condiciones complejas de normas utilizando AND, OR, NOT, así como la agrupación con paréntesis. El generador de subexpresiones valida la norma al aplicar los cambios y avisa si hay paréntesis desemparejados y operadores lógicos a los que no se les da soporte.
  • Colección de políticas de salud
    Utilice esta página para crear y modificar políticas de salud. Las políticas de salud dan soporte a dos aproximaciones básicas para mantener un entorno "saludable": la detección y la prevención.
  • Valores de política de salud
    Utilice esta página para modificar políticas de salud que ya existen. Las políticas de salud permiten mantener un entorno "saludable" mediante la utilización de metodologías de detección y prevención.
