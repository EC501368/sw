Intelligent Management para plug-ins del servidor web

Utilice esta página para habilitar y configurar Intelligent Management para servidores web Apache o IBM HTTP.

Para ver la página de la consola administrativa, pulse Servidores > Tipos de servidores > Servidores web > nombre_servidor_web > Intelligent Management.

Habilitar

Seleccione esta casilla de verificación para habilitar las funciones de Intelligent Management para el servidor web.

Identificador de célula

Especifica el identificador de célula exclusivo de la célula local.

Servicio de Intelligent Management

Especifica los parámetros de conexión del servicio Intelligent Management.

Información Definición
Intervalo de reintento Especifica el intervalo de tiempo (en segundos) entre intentos de conexión.
Número máximo de reintentos Especifica el número máximo de reintentos para habilitar el servicio Intelligent Management.

Células remotas

Puede añadir o suprimir células remotas o puede editar o renovar las propiedades de células remotas. Algunas acciones tienen opciones.

Botón Acción resultante
Añadir Inicia el asistente para añadir una célula remota.
Suprimir Suprime las células remotas seleccionadas. De forma opcional, seleccione la casilla de verificación Suprimir certificados que se muestra más abajo para eliminar los certificados.
Editar Inicia el asistente para editar las propiedades de las células remotas en la sección Intelligent Management del archivo plugin-cfg.xml.
Renovar Renueva las propiedades de células remotas de la célula seleccionada. Si se ha habilitado la seguridad administrativa, proporcione el nombre de usuario y la contraseña en el campo ID de usuario (Renovar) y Contraseña (Renovar).
Información Definición
Host Especifica el nombre de host de la célula remota.
ID de célula Especifica un identificador de célula exclusivo.
Habilitado Indica si se han habilitado las funciones de Intelligent Management para la célula remota especificada.
Conectores Especifica los conectores de la célula remota.

Importar certificados (Importar)

Seleccione la seleccionar para importar certificados de célula remota. Esta casilla de verificación está seleccionada de forma predeterminada.

Nota: Para completar el intercambio de certificados para habilitar el plug-in de Intelligent Management con el fin de conectar de forma segura con una célula remota, podría ser necesario importar el certificado de la célula en el almacén de claves de células remotas. Puede utilizar el mandato de script AdminTask.retrieveSignerFromPort o la consola administrativa pulsando Certificado SSL y gestión de claves > Almacenes de claves y certificados > CellDefaultTrustStore > Certificados de firmante > Recuperar de puerto.

Suprimir certificados (Suprimir)

Seleccione la casilla de verificación para suprimir certificados de las células.

ID de usuario (Renovar)

Escriba el ID de usuario de la célula seleccionada para renovar.

Contraseña (Renovar)

Escriba la contraseña de la célula seleccionada para renovar.
