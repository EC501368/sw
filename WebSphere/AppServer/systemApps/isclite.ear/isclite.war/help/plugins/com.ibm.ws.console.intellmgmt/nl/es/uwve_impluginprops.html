Propiedad de plug-in de Intelligent Management

Utilice esta página para configurar una propiedad de Intelligent Management para el plug-in de servidor web.

Para ver la página de la consola administrativa, pulse Servidores > Tipos de servidor > Servidores web > nombre_servidor_web > Intelligent Management > Propiedades de plug-in de Intelligent Management > Nuevo.

También puede ver esta página de la consola si pulsa Servidores > Tipos de servidor > Servidores web > nombre_servidor_web > Intelligent Management > Propiedades de plug-in de Intelligent Management > propiedad_plug_in.

Nombre

Especifique el nombre de propiedad de Intelligent Management. El nombre es una clave de propiedad que se utiliza para establecer las propiedades de configuración internas del sistema.

Valor

Especifique el valor de la propiedad de Intelligent Management. El valor es una serie que se utiliza para establecer las propiedades de configuración internas del sistema.
