Recopilación de propiedades de Intelligent Management

Utilice esta página para configurar una célula remota para el plug-in de servidor web de Intelligent Management.

Para ver la página de la consola administrativa, pulse Servidores > Tipos de servidor > Servidores web > nombre_servidor_web > Intelligent Management > Propiedades de plug-in de Intelligent Management.

Operaciones de propiedades de Intelligent Management

Puede añadir o suprimir propiedades, o puede modificar las propiedades de células existentes.

Botón Acción resultante
Nuevo Lanza el asistente para añadir una propiedad nueva de Intelligent Management.
Suprimir Suprime las propiedades seleccionadas.
Información Definición
Nombre Clave de la propiedad que se va a utilizar para las propiedades de configuración internas del sistema.
Valor Valor de serie que se va a utilizar para las propiedades de configuración internas del sistema.
