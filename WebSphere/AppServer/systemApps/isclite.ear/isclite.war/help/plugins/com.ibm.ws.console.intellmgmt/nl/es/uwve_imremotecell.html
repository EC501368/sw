Célula remota para el servidor web de Intelligent Management

Utilice esta página para configurar una célula remota para el plug-in de servidor web de Intelligent Management.

Para ver esta página de la consola administrativa, pulse Servidores > Tipos de servidor > Servidores web > nombre_servidor_web > Intelligent Management > Nuevo.

Habilitar

Seleccione la casilla de verificación para habilitar las funciones de Intelligent Management para la célula remota.

Host

Especifica el nombre de host de la célula remota.

Puerto

Especifica el puerto de SOAP de la célula remota.

Identificador de célula

Especifica un identificador de célula remota exclusivo.

ID de usuario

Especifica el ID de usuario del administrador de la célula remota.

Contraseña

Especifica la contraseña de usuario del administrador de la célula remota.

Importar certificados

Seleccione la seleccionar para importar certificados de célula remota. Esta casilla de verificación está seleccionada de forma predeterminada.
