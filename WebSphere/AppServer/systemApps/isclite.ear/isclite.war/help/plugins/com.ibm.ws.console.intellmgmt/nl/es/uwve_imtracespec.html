Especificación de rastreo

Utilice esta página para configurar el rastreo para el plug-in de servidor web.

Para ver esta página de la consola administrativa, pulse Servidores > Tipos de servidor > Servidores web > nombre_servidor_web > Intelligent Management > Especificación de rastreo.

Rastreo

Establezca la especificación de rastreo e indique un nivel de detalle de registro. Expanda el árbol de especificaciones de rastreo y pulse una especificación de rastreo para seleccionar un nivel de detalle de registro.
