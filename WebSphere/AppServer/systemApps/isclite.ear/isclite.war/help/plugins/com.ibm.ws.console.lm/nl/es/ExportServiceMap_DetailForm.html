Correlaciones de servicios - Exportar

Utilice este panel para exportar una o varias correlaciones de servicios.

Para ver este panel en la consola administrativa, pulse Integración de servicios > Correlación de servicios > Correlaciones de servicio > Exportar.

Este panel muestra las siguientes propiedades:

Nombre
Muestra el nombre de la correlación de servicios o las correlaciones de servicios que ha seleccionado para exportar.
Nombre de biblioteca de correlación de servicios
Una biblioteca es un contenedor de las correlaciones de servicios que ha seleccionado para exportar. Especifique un nombre para la biblioteca que desee crear.

Botones

Etiqueta Acción
Exportar Confirma la selección y abre un recuadro de diálogo Guardar.
Cancelar Cancelar los cambios realizados y, a continuación, volver al panel anterior.

Los enlaces marcados (en línea) requieren acceso a Internet. Cada enlace inicia una búsqueda del tema en el Information Center. Cuando se listen los resultados de la búsqueda, seleccione el tema que se ajuste mejor a la configuración.
