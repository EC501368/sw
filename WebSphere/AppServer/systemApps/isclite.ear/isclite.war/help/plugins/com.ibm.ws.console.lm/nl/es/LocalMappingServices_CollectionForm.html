Colección de servicios de correlaciones locales

Un servicio de correlaciones local define el punto de intercepción entre un cliente de servicios y un proveedor de servicios.

Para ver este panel en la consola administrativa, pulse Integración de servicios > Correlación de servicios > Servicios de correlaciones locales

  • Para examinar las propiedades de un servicio de correlaciones local, seleccione su nombre en la lista.
  • Para actuar sobre uno o varios de los servicios de correlaciones locales, pulse los recuadros de selección que aparecen junto a los nombres de los elementos sobre los que desea actuar y, a continuación, utilice el botón que se proporciona.
  • Para cambiar los servicios de correlaciones locales que se enumeran, o para cambiar el nivel de detalle que se muestra para dichas entradas, utilice los valores de Filtro.
  • Para crear un nuevo servicio de correlaciones local, pulse Nuevo.

Este panel muestra las siguientes propiedades:

Nombre
Nombre exclusivo de este servicio de correlaciones local.
Descripción
Descripción de este servicio de correlaciones local.
Correlación de servicios conectada
La correlación de servicios que utiliza el servicio de correlaciones local.
Estado
El estado del servicio de correlaciones local: Iniciado, representado por una flecha verde (), Detenido, representado por una cruz roja () o Desconocido, representado por una interrogación (?).
Tabla 1. Estado y comportamiento de un servicio de correlaciones local
Estado de Servicio de correlaciones local ¿Conectado a una correlación de servicios? Sí/No Comportamiento
Iniciado No La solicitud se direcciona al servicio de destino original especificado por el cliente de servicio.
Detenido No El servicio no está disponible. Se devuelve un error.
Iniciado Sí Se aplica la correlación de servicios.
Detenido Sí El servicio no está disponible. Se devuelve un error.

Botones

Etiqueta Acción
Nuevo Crear un servicio de correlaciones local.
Suprimir Suprimir uno o varios servicios de correlaciones local. Puede suprimir un servicio de correlaciones local sólo si no está conectado a una correlación de servicios.
Conectar Conectar el servicio de correlaciones local seleccionado a una correlación de servicios.
Desconectar Desconectar uno o varios servicios de correlaciones locales de una correlación de servicios.
Iniciar Iniciar uno o varios servicios de correlaciones locales.
Detener Detener uno o varios servicios de correlaciones locales.
