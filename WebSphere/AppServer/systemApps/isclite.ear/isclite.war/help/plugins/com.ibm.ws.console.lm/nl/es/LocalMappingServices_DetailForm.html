Valores del servicio de correlaciones local

Un servicio de correlaciones local define el punto de intercepción entre un cliente de servicios y un proveedor de servicios. Utilice este panel para establecer las propiedades de un servicio de correlaciones local específico.

Para ver este panel en la consola administrativa, pulse Integración de servicios > Correlación de servicios > Servicios de correlaciones locales > nombre_definición_servicio_correlaciones_locales

Este panel muestra las siguientes propiedades:

Propiedades generales

Nombre
Nombre exclusivo de este servicio de correlaciones local.
Nota: Los nombres deben ser exclusivos dentro de una célula y no pueden contener caracteres no admitidos. No están soportados los caracteres siguientes:
  • /\,#$@:;"*?<>|=+&%'
  • (espacio)
  • la combinación de caracteres ]]>
Descripción
Descripción de este servicio de correlaciones local. Este campo es opcional.

Propiedades de servicio

Dirección de punto final
Dirección de punto final del proveedor de servicios utilizado por la solicitud de servicio web que intercepta el servicio de correlaciones local.
Tipo de puerto
Tipo de puerto asociado con la solicitud de servicio interceptada por el servicio de correlaciones local. Este campo es opcional.
Espacio de nombres
Espacio de nombres del servicio y puerto.
Nombre de servicio
Nombre de la solicitud de servicio web interceptada por el servicio de correlaciones local.
Nombre de puerto
Nombre de puerto asociado con la solicitud de servicio web interceptada por el servicio de correlaciones local.
Nota: Los valores de los campos en este panel deben coincidir con los valores de la correlación de servicios que está conectada con el servicio de correlaciones local.

Correlaciones de servicios

Correlación de servicios conectada
La correlación de servicios que está conectada con este servicio de correlaciones local, que se muestra en un menú desplegable. El menú puede utilizarse para seleccionar una correlación de servicios instalada diferente para conectarse con este servicio de correlaciones local. El menú sólo muestra si se ha instalado al menos una correlación de servicios. Si no se han instalado correlaciones de servicios, aparece un mensaje informativo.

Si se ha conectado una correlación de servicios al servicio de correlaciones local seleccionado y selecciona una correlación de servicios en este menú distinta de la que está conectada, la correlación de servicios original se desconecta cuando se conecta la nueva correlación de servicios.

Nota: Sólo se puede conectar una correlación de servicios a un servicio de correlaciones local. Sin embargo, una correlación de servicios pueden conectarse a varios servicios de correlaciones locales.
