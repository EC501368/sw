Colección de correlaciones de servicios

Una correlación de servicios define cómo se transforma o redirecciona una solicitud que se envía desde un cliente de servicios a un proveedor de servicios.

Para ver este panel en la consola administrativa, pulse Integración de servicios > Correlación de servicios > Correlaciones de servicio

  • Para examinar las propiedades de una correlación de servicios, seleccione su nombre en la lista.
  • Para actuar sobre una o varias de las correlaciones de servicios, pulse los recuadros de selección que aparecen junto a los nombres de los elementos sobre los que desea actuar y, a continuación, utilice el botón que se proporciona.
  • Para cambiar las correlaciones de servicios que se enumeran, o para cambiar el nivel de detalle que se muestra para dichas entradas, utilice los valores de Filtro.

Este panel muestra las siguientes propiedades:

Nombre
Nombre exclusivo de esta correlación de servicios.
Descripción
Descripción de esta correlación de servicios.
Servicios de correlaciones locales conectados
El servicio de correlaciones local al que está conectada esta correlación de servicios. Pulse Ver todos... para ver todos los servicios de correlaciones locales conectados a esta correlación de servicios.

Botones

Etiqueta Acción
Instalar Instalar una o varias correlaciones de servicios.
Desinstalar Desinstalar una o varias correlaciones de servicios seleccionadas.
