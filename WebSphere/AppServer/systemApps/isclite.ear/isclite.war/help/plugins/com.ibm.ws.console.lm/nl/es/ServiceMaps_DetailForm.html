Valores de correlación de servicios

Una correlación de servicios define cómo se transforma o redirecciona una solicitud que se envía desde un cliente de servicios a un proveedor de servicios. Utilice este panel para ver las propiedades de una correlación de servicios específico.

Para ver este panel en la consola administrativa, pulse Integración de servicios > Correlación de servicios > Correlaciones de servicio > nombre_definición_correlación_servicios.

Este panel muestra las siguientes propiedades:

Propiedades generales

Nombre
Nombre exclusivo de esta correlación de servicios.
Descripción
Descripción de esta correlación de servicios.

Propiedades de aplicación de nivel empresarial

Nombre
El nombre de la aplicación de nivel empresarial utilizada por esta correlación de servicios
Estado
El estado de la aplicación de nivel empresarial utilizada por esta correlación de servicios. Los valores posibles son: Iniciado, Detenido, Inicio parcial o Desconocido. Para obtener más información sobre los estados de aplicación de nivel empresarial, consulte Colección de aplicaciones de nivel empresarial (en línea).
Desplegada en los servidores
Lista los nombres de los servidores en los que se ha desplegado esta correlación de servicios.

Propiedades de servicio

Tipo de puerto
Tipo de puerto asociado con la solicitud de servicio interceptada por el servicio de correlaciones local. Este campo es opcional.
Espacio de nombres
Espacio de nombres del servicio y puerto.
Nombre de servicio
Nombre de la solicitud de servicio web interceptada por el servicio de correlaciones local.
Nombre de puerto
Nombre de puerto asociado con la solicitud de servicio web interceptada por el servicio de correlaciones local.
Nota: Los valores de los campos en este panel deben coincidir con los valores del servicio de correlaciones local que está conectado con esta correlación de servicios.

Servicios de correlaciones locales conectados

Esta lista muestra los servicios de correlaciones locales que están conectados con esta correlación de servicios. Si no hay servicios de correlaciones locales conectados con esta correlación de servicios, se muestra un mensaje informativo.
