Paneles de la consola administrativa de la correlación de servicios

Puede utilizar estos paneles de la consola administrativa para gestionar los recursos de la correlación de servicios.

Los enlaces marcados (en línea) requieren acceso a Internet. Cada enlace inicia una búsqueda del tema en el Information Center. Cuando se listen los resultados de la búsqueda, seleccione el tema que se ajuste mejor a la configuración.

Compruebe si hay actualizaciones para este tema (en línea)

  • Colección de servicios de correlaciones locales
    Un servicio de correlaciones local define el punto de intercepción entre un cliente de servicios y un proveedor de servicios.
  • Valores del servicio de correlaciones local
    Un servicio de correlaciones local define el punto de intercepción entre un cliente de servicios y un proveedor de servicios. Utilice este panel para establecer las propiedades de un servicio de correlaciones local específico.
  • Servicios de correlaciones locales - Conectar
    Utilice este panel para conectar una correlación de servicios a un servicio de correlaciones local.
  • Colección de correlaciones de servicios
    Una correlación de servicios define cómo se transforma o redirecciona una solicitud que se envía desde un cliente de servicios a un proveedor de servicios.
  • Valores de correlación de servicios
    Una correlación de servicios define cómo se transforma o redirecciona una solicitud que se envía desde un cliente de servicios a un proveedor de servicios. Utilice este panel para ver las propiedades de una correlación de servicios específico.
  • Correlaciones de servicios - Instalar
    Utilice este panel para especificar la biblioteca de correlaciones de servicios que contiene las correlaciones de servicios que desea instalar.
  • Correlaciones de servicios - Desinstalar
    Utilice este panel para desinstalar una o varias correlaciones de servicios.
