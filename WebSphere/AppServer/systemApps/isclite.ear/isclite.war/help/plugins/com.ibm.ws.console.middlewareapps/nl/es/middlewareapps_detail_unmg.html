Instalación de aplicaciones web no gestionadas

Este asistente sirve para instalar aplicaciones web no gestionadas.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Seleccionar una aplicación web no gestionada > Instalar una nueva aplicación de middleware > Siguiente.

Paso 1: Definir propiedades generales de la clase de trabajo

Se utiliza para definir las propiedades generales de la aplicación web no gestionada.

Nombre

Especifica un nombre para la aplicación. El nombre de una aplicación no gestionada de middleware debe ser único entre todas las aplicaciones en la célula.

El campo de nombre no puede contener los caracteres siguientes: # \ / , : ; " * ? < > | = + & % ' . El nombre no puede empezar con un punto (.) o un espacio. Un espacio no genera un error, pero los espacios al principio o al final se eliminan automáticamente.

El nombre debe ser distinto de cualquier otro nombre de aplicación que se haya definido en la célula. Por ejemplo en una misma célula una aplicación web no gestionada no puede tener el mismo nombre que una aplicación Java EE (Java Platform, Enterprise Edition).

Edición

Especifica la edición actual de la aplicación.

Descripción

Campo opcional para especificar la descripción de la aplicación.

Paso 2: Definir las propiedades de despliegue

Nombre de módulo

Especifica la vía de acceso local o remota completa en que se encuentra la aplicación.

Raíz de contexto

Se utiliza con módulos web (archivos .war) y módulos SIP (archivos .sar).

Host virtual

Es necesario para todas las aplicaciones.

Destinos de despliegue

Especifica el destino en que se instalará la aplicación no gestionada.

Tabla 1. Botones de acción de destino de despliegue
Botón Acción resultante
Añadir Seleccione los destinos de despliegue que desee añadir a la lista de módulos y, a continuación, pulse Añadir.
Eliminar Seleccione los destinos de despliegue que desee suprimir de la aplicación o la representación de la aplicación del repositorio de configuración y, a continuación, pulse Eliminar.
Aplicar Después de añadir los destinos de despliegue, pulse Aplicar para añadir los destinos de despliegue seleccionados a la lista de módulos.
Nueva Permite crear un nuevo módulo web dentro de este panel.

Confirmar la nueva representación de la aplicación

Confirma el nombre, la edición y la descripción de la aplicación que se desea añadir.

Pulse Finalizar para confirmar la instalación de la aplicación de middleware.
