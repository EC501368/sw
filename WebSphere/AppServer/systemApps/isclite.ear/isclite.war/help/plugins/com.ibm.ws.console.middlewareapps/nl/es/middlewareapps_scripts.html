Valores de scripts de aplicación

Utilice esa página para cambiar los scripts de configuración y limpieza de su aplicación de middleware.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Todas las aplicaciones > nombre_aplicación_middleware > Scripts de aplicación.

Se pueden especificar los scripts que se utilizarán desde un sistema de archivos local o desde un sistema de archivos remoto.

Script de configuración

Especifica la vía de acceso al script de configuración de su aplicación.

Script de limpieza

Especifica la vía de acceso al script de limpieza de su aplicación.
