Descriptores de middleware

Vaya a los temas de ayuda de esta sección.

  • Colección de descriptores de middleware
    Los descriptores de plataforma de middleware proporcionan información sobre las plataformas de middleware gestionadas a los sensores de descubrimiento y al entorno de tiempo de ejecución de Intelligent Management.
  • Detalles de descriptores de middleware
    Los descriptores de plataforma de middleware proporcionan información sobre las plataformas de middleware gestionadas a los sensores de descubrimiento y al entorno de tiempo de ejecución de Intelligent Management.
  • Valores de detalle de versión
    Utilice esta página para cambiar los valores específicos a una determinada versión de un descriptor de middleware. Por ejemplo, si se están ejecutando dos versiones distintas de PHP, se podrían cambiar estos valores para especificar ubicaciones de instalación para cada una de las versiones.
