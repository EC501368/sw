Nodos de middleware

Vaya a los temas de ayuda de esta sección.

  • Autenticación
    Si desea iniciar un nodo es necesario proporcionar un nombre de usuario y una contraseña para el nodo.
  • Valores de nodo de middleware
    Esta página sirve para definir o cambiar la configuración de un nodo de middleware.
  • Colección de nodos de middleware
    Los nodos de middleware incluyen a todos los nodos de su configuración. Estos nodos pueden incluir nodos que tienen instalada una versión del producto nodos que están ejecutando otros tipos de servidores.
