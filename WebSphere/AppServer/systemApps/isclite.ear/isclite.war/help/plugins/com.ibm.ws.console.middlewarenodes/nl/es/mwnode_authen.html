Autenticación

Si desea iniciar un nodo es necesario proporcionar un nombre de usuario y una contraseña para el nodo.

Para acceder a esta página de la consola administrativa, pulse Administración del sistema > Nodos de middleware. Seleccione un nodo y, a continuación, seleccione la operación Iniciar agente. Pulse Ejecutar.

Si se desea guardar el nombre de usuario y la contraseña, existe la posibilidad de escribirlas en panel de la consola administrativa de destinos de instalación. Pulse Administración del sistema > Gestor de instalación centralizada > Destinos de instalación. Añada el nodo como un destino de instalación, e incluya el nombre de usuario y la contraseña.

Nombre de usuario

Especifica el nombre de usuario que se utiliza para iniciar una sesión en la máquina remota.

Contraseña

Especifica la contraseña que se utiliza para iniciar una sesión en la máquina remota.
