Valores de nodo de middleware

Esta página sirve para definir o cambiar la configuración de un nodo de middleware.

Para ver esta página de la consola administrativa, pulse Administración del sistema > Nodos de middleware > nombre_nodo_middleware.

Nombre

Especifica un nombre del nodo de middleware. El nombre debe ser exclusivo entre todos los nodos en la célula. En esta página, no se puede editar el valor de nombre de nodo.

Normalmente, un nombre de nodo es el mismo que el nombre de host del sistema. Sin embargo, existe la posibilidad de elegirlo. Se puede hacer que el nombre de nodo sea distinto del nombre de host.

Nombre de host

Especifica el nombre de host del nodo de middleware. En esta página, no se puede editar el valor de nombre de host.

Tipo de plataforma

Especifica el sistema operativo en que se ejecuta el nodo de middleware.

Elija entre los siguientes tipos de plataforma:
  • Windows
  • AIX®
  • HP-UX
  • Solaris
  • Linux
  • OS/400®
  • z/OS®
