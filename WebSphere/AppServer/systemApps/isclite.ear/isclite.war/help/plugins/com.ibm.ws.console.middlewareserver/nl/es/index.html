Servidores de middleware

Vaya a los temas de ayuda de esta sección.

  • Creación de un servidor de middleware
    Esta página sirve para crear un servidor de middleware. Los servidores de middleware son servidores del nivel de middleware que alojan aplicaciones o sus datos.
  • Configuración externa
    Esta página sirve para editar un archivo de configuración para un servidor de middleware.
  • Servicio de configuración externo
    Esta página sirve para habilitar o inhabilitar archivos de configuración asociados con un servidor de middleware.
  • Servicio de anotaciones cronológicas externas
    Esta página sirve para acceder a los archivos de anotaciones cronológicas de los servidores de middleware.
  • Visor de registro de anotaciones cronológicas
    Esta página sirve para ver los archivos de anotaciones cronológicas de los servidores de middleware.
  • Colección de servidores de middleware
    Utilice esta página para ver y gestionar servidores de middleware. Los servidores de middleware son servidores genéricos en el nivel de middleware que alojan aplicaciones o sus datos.
  • Valores de operaciones de servidor de middleware
    Esta página sirve para configurar operaciones para servidores de middleware. Puede utilizar las operaciones de servidor para ejecutar archivos ejecutables en los servidores de middleware desde la consola administrativa.
  • Colección de plantillas de servidor de middleware
    Esta página sirve para crear una nueva plantilla de servidor. Las plantillas de servidor definen valores de configuración al crear servidores de middleware.
  • Valores de plantilla de servidor de middleware
    Esta página sirve para configurar los valores de una nueva plantilla de servidor que haya creado. Los tipos de servidor que existen para crear una plantilla incluyen a servidor de aplicaciones, genérico, direccionador On Demand, proxy o servidor web. También se pueden crear o añadir plantillas de servidor para un servidor de middleware que ya exista en su entorno como, por ejemplo, PHP, servidor HTTP personalizado o WebSphere Application Server Community Edition.
  • Valores de detalle de servidor de middleware
    Esta página sirve para ver la configuración de los servidores de middleware.
  • Operaciones de servidor
    Esta página permite añadir, editar y suprimir operaciones de servidor.
