Servicio de configuración externo

Esta página sirve para habilitar o inhabilitar archivos de configuración asociados con un servidor de middleware.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > servidor_middleware > Servicio de configuración externo o Servidores > Tipos de servidor > tipo_servidor_middleware > servidor_middleware > Servicio de configuración externo.

Hay que poseer privilegios administrativos para configurar el servicio y para ver y editar los archivos. Los usuarios con privilegios de configurador pueden utilizar el servicio para editar y ver los archivos, mientras que los usuarios con privilegios de supervisor sólo pueden leerlos.

El servicio de configuración externo se utiliza con los siguientes propósitos:
  • Hacer referencia archivos de configuración externos de forma que se pueda modificar la configuración duplicada necesaria para habilitar el servidor en su entorno On Demand.
  • Gestionar los servidores de middleware que ya existen desde una ubicación central, la consola administrativa.

Habilitar servicio de configuración externo

Habilita el servicio de configuración externo.

Seleccione el recuadro de selección que permite habilitar el servicio. Pulse Aceptar.

Añadir un archivo al servicio de configuración externo

Se debe especificar la vía de acceso al archivo así como sus permisos para añadirlo al servicio de configuración externo. Pulse Añadir para añadir el archivo a la lista.

Nombre de vía de acceso completa de un archivo de configuración
Especifica en nombre de vía de acceso completa del archivo de configuración. Existe la posibilidad de utilizar una variable en el nombre de la vía de acceso, por ejemplo: 
${CATALINA_HOME}/conf/server.xml
Grabable
Si se selecciona, se puede grabar en el archivo desde la consola administrativa. Si no se selecciona, el archivo es de sólo lectura.

Lista de archivos de configuración existentes

Muestra una lista de los archivos de configuración que se han añadido al servicio de configuración externo para su servidor.

Vía de acceso de archivo
Identifica la vía de acceso y el nombre del archivo de configuración. La vía de acceso tiene en una lista de nombres de archivo por omisión.
Privilegios de archivo
Especifica los privilegios de grabación o lectura que se asignan a cada archivo. Seleccione un archivo y, a continuación, pulse Conmutar privilegios para cambiar el tipo de privilegio.
