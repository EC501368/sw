Configuración externa

Esta página sirve para editar un archivo de configuración para un servidor de middleware.

Para ver esta página de la consola administrativa, pulse Servidores > Tipos de servidor > tipo_servidor_middleware > servidor_middleware > Configuración externa o Servidores > Todos los servidores > servidor_middleware > Configuración externa.

Configuración externa

Muestra los archivos de configuración que se pueden editar.

Si desea cambiar la configuración de una plantilla de servidor PHP, edite los archivos httpd.conf y php.ini. El archivo httpd.conf es el archivo de configuración del servidor Apache. Además, existe la posibilidad de definir variables en este archivo para utilizarlas en la consola administrativa de WebSphere Application Server. El archivo php.ini contiene datos relacionados con el módulo PHP.

Recuperar

Recupera del servidor el archivo seleccionado.

Aplicar

Aplica los cambios realizados al archivo.

Restablecer

Restablece el archivo a su formato original antes de que realizasen cambios o antes de que pulsase Aplicar.
