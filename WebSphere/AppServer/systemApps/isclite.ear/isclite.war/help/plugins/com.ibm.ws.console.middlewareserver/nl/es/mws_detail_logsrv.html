Servicio de anotaciones cronológicas externas

Esta página sirve para acceder a los archivos de anotaciones cronológicas de los servidores de middleware.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > servidor_middleware > Servicio de anotaciones cronológicas externas o Servidores > Tipos de servidor > tipo_servidor_middleware > servidor_middleware > Servicio de anotaciones cronológicas externas.

Hay que poseer privilegios administrativos para configurar el servicio y ver los archivos de anotaciones cronológicas. Los usuarios con privilegios de configurador pueden utilizar el servicio únicamente para ver los archivos.

El servicio del visor de anotaciones cronológicas externas se configura para ver en la consola administrativa dichas anotaciones para los servidores de middleware. Los directorios se configuran para permitir visualizar todos los archivos en el directorio mediante el servicio de visualización de anotaciones cronológicas. Las plantillas de servidor proporcionan una configuración por omisión de la lista de directorios de anotaciones cronológicas para los servidores de dicho tipo. Los usuarios con el rol de administrador pueden añadir o eliminar directorios de dicha lista.

Habilitar servicio de visualización de anotaciones cronológicas remotas

Habilita el servicio de visualización de anotaciones cronológicas

Seleccione el recuadro para habilitar el servicio de visualización de anotaciones cronológicas. En el campo que se proporciona, escriba el nombre del directorio para habilitar su visualización en el servicio de visualización de anotaciones cronológicas externas. Pulse Añadir para añadir el directorio a la lista de directorios en la tabla que hay continuación.
