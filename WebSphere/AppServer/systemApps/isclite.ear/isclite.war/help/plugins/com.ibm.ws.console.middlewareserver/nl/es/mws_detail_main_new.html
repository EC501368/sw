Creación de un servidor de middleware

Esta página sirve para crear un servidor de middleware. Los servidores de middleware son servidores del nivel de middleware que alojan aplicaciones o sus datos.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > Nuevo.

Hay que poseer privilegios de administrador para crear servidores de middleware. Consulte la sección de referencia en el centro de información para obtener información sobre otros roles administrativos y privilegios.

Añadir un servidor

Permite crear una nueva instancia de servidor o añadir un servidor de middleware que ya exista. Estos servidores tienen una gestión completa del ciclo de vida.

Si se creó un servidor de middleware fuera de la consola administrativa, puede instalar el agente de nodo en dicho servidor y añadirlo a la célula. Esta acción crea una representación del servidor en la consola administrativa que se puede utilizar para gestionar el servidor que se ha creado de forma externa. Estos servidores tienen una gestión asistida del ciclo de vida.

Crear una nueva instancia de servidor:
  • Servidor de aplicaciones
  • Servidor genérico
  • Direccionador On Demand
  • Servidor PHP
  • Servidor proxy
  • Servidor web
  • Servidor WebSphere Application Server Community Edition
Añadir un servidor existente en la célula:
  • Servidor HTTP personalizado
  • Servidor BEA WebLogic
  • Servidor JBoss
  • Servidor PHP
  • Servidor Apache Tomcat
  • Servidor de aplicaciones de WebSphere externo
  • Servidor WebSphere Application Server Community Edition
  • Servidor de perfiles Liberty

Seleccionar un nodo

Seleccione un nodo que corresponda al servidor que desea crear y defina un nombre de servidor.

Seleccionar un tiempo de ejecución

Para servidores de WebSphere Application Community Edition, elija un entorno de tiempo de ejecución para su servidor.

Seleccionar una plantilla de servidor

Seleccione la plantilla que especifique mejor los atributos del servidor que desea crear.

Las plantillas de servidor son para los distintos tipos de servidor. Existe la posibilidad de crear plantillas de servidor propias. Para crear su propia plantilla de servidor, pulse Servidores > Todos los servidores > Plantillas.

Confirmar el nuevo servidor de middleware

El campo de resumen de acciones describe las elecciones realizadas con el asistente. Pulse Finalizar para crear el nuevo servidor de middleware.
