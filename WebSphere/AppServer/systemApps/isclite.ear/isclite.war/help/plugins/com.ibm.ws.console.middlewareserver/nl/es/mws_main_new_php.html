Creación de un servidor de middleware PHP

Esta página sirve para crear un nuevo servidor de middleware PHP.

Pulse Servidores > Servidores de middleware > Nuevo > Crear una nueva instancia de servidor > Servidor PHP para ver esta página de la consola administrativa.

Crear una nueva instancia de servidor o Añadir un servidor existente

Este panel del asistente sirve para crear una nueva instancia de servidor, o para añadir un servidor existente al entorno de WebSphere Extended Deployment. Al crear una nueva instancia de servidor creará un nuevo servidor, mientras que añadir a un servidor existente añade un servidor de aplicaciones existente que no es de WebSphere al entorno de WebSphere Application Server.

Paso 1: Añadir un servidor
Crea una nueva instancia de servidor que es exclusiva en la célula.
Elija un servidor PHP de la lista:
  • Aplicación
  • Genérico
  • Direccionador On Demand
  • PHP
  • Proxy
  • Web
Añadir un servidor existente en la célula:
Elija un tipo de servidor de la lista:
  • CustomHTTP
  • BEA Weblogic Server
  • JBoss
  • PHP
  • Apache Tomcat
  • WASCE
Paso 2: Seleccionar un nodo

Seleccione un nodo que corresponda al servidor que desea crear y defina un nombre de servidor.

Paso 3: Seleccionar un entorno de tiempo de ejecución Apache

Seleccione un entorno de tiempo ejecución Apache para su servidor PHP.

Debajo de cada versión se mostrarán las correspondientes vías de acceso.

Paso 4: Seleccionar un entorno de tiempo de ejecución PHP

Seleccione un entorno de tiempo de ejecución PHP para el servidor.

Debajo de cada versión se mostrarán las correspondientes vías de acceso.

Paso 5: Seleccionar una plantilla PHP

Seleccione una plantilla para el servidor PHP.

Elija una plantilla por omisión de la lista. La plantilla PHP incluye la plantilla tanto para las versiones PHP como Apache que especificó en el paso anterior.

Paso 6: Confirmar nuevo servidor PHP

Pulse Finalizar para crear el nuevo servidor PHP.

El campo de resumen de acciones describe las elecciones realizadas con el asistente.
