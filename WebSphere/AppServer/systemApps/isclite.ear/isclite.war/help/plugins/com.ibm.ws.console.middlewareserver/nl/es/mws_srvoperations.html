Operaciones de servidor

Esta página permite añadir, editar y suprimir operaciones de servidor.

Las operaciones de servidor consisten en definiciones de proceso Java y no Java que se definen para ejecutarlas en servidores de middleware. Estas operaciones se crean para realizar tareas como las que habilitan o inhabilitan el rastreo, inician o detienen aplicaciones o que consultan el estado de ejecución del servidor.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > servidor_middleware > Operaciones de servidor o seleccioneServidores > Tipos de servidor > tipo_servidor_middleware > servidor_middleware > Operaciones de servidor.

Nombre

Especifica el nombre de la operación de servidor.

Pulse sobre el nombre para ver la configuración de operación del servidor.

SO soportados

Identifica al sistema operativo al que se da soporte.

Acción

Identifica la acción ejecutable que se asocia con la operación de servidor.
