Valores de operaciones de servidor de middleware

Esta página sirve para configurar operaciones para servidores de middleware. Puede utilizar las operaciones de servidor para ejecutar archivos ejecutables en los servidores de middleware desde la consola administrativa.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > servidor_middleware > Operaciones de servidor > operación_servidor o seleccioneServidores > Tipos de servidor > tipo_servidor_middleware > servidor_middleware > Operaciones de servidor > operación_servidor.

Nombre

Especifica el nombre del servidor.

Ejecutable

Nombre del archivo ejecutable.

Argumentos ejecutables

Hace referencia a los nombres y los valores de variable como argumentos ejecutables.

Cada ejecutable debe estar en su propia línea. Cuando el agente de nodo invoca la operación, se inserta el valor para la variable.

Nombre de una variable para hacer referencia a un nombre de usuario en argumentos utilizados en la ejecución

Especifica una variable para hacer referencia a un nombre de usuario.

Nombre de usuario a sustituir para la variable de nombre de usuario en el momento de la invocación

Especifica un nombre de usuario para almacenar de forma cifrada en la configuración y al que se puede hacer referencia, junto con la contraseña cifrada, como un argumento ejecutable en la operación.

Nombre de una variable para hacer referencia a una contraseña en argumentos utilizados en la ejecución

Especifica una variable para hacer referencia a una contraseña.

Contraseña a sustituir para la variable de contraseña en el momento de la invocación

Especifica una contraseña para almacenar de forma cifrada en la configuración y a la que se puede hacer referencia, junto con el nombre de usuario cifrado, como un argumento ejecutable en la operación.

Directorio de trabajo

Identifica el directorio de trabajo para la operación de servidor.

SO soportados

Especifica los sistemas operativos asociados con la operación de servidor.

Nombre de una variable de entorno que almacena un nombre de archivo PID

Especifica la variable de entorno que almacena el nombre de archivo PID.

El nombre del archivo PID se almacena en una variable de WebSphere. El nombre del archivo PID especifica la ubicación del archivo PID en que se ejecuta la operación de servidor.
