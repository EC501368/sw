Valores de servidor PHP

Utilice esta página para gestionar y visualizar información de servidores PHP (PHP Hypertext Preprocessor). Los servidores PHP están definidos en WebSphere Virtual Enterprise como un servidor Apache con el módulo mod_php. La configuración de servidores PHP permite gestionar aplicaciones PHP en el mismo entorno administrativo que las aplicaciones J2EE (Java 2 Platform, Enterprise Edition).

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > servidor_php o Servidores > Otros servidores de middleware > servidores PHP > servidor_php

Nombre
Nombre de nodo

Especifica el nombre de nodo asociado con el servidor. Normalmente, un nombre de nodo es el mismo que el nombre de host del sistema. Sin embargo, existe la posibilidad de elegirlo. Se puede hacer que el nombre de nodo sea distinto del nombre de host.

Ubicación raíz del servidor PHP

Especifica la vía de acceso al entorno de tiempo de ejecución PHP.

Ubicación raíz del servidor Apache

Especifica la vía de acceso al entorno de tiempo de ejecución Apache.

Versión PHP

Especifica la versión del entorno de tiempo de ejecución del servidor PHP.

Las versiones de PHP a las que se da soporte incluyen: 4.x y 5.x.

Versión Apache

Especifica la versión del entorno de tiempo de ejecución del servidor Apache.

Las versiones de servidores Apache a las que se da soporte incluyen: 1.3.x, 2.0.x y 2.2.x.

Conector HTTP

Especifica las propiedades específicas del servidor, incluidos el número de puerto y la habilitación.

Según sea el tipo de servidor, se pueden mostrar otros campos. Por ejemplo, los servidores PHP tienen campos para la ubicación raíz de Apache Tomcat y PHP. Los conectores de HTTP y HTTPS no se muestran con todos los tipos de servidor.

Conector HTTPS

Especifica las propiedades específicas del servidor, incluidos el número de puerto, la habilitación y la seguridad.

Los conectores de HTTP y HTTPS no se muestran con todos los tipos de servidor.
