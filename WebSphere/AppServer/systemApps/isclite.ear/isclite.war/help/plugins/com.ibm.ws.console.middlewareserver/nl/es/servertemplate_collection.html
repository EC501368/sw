Colección de plantillas de servidor de middleware

Esta página sirve para crear una nueva plantilla de servidor. Las plantillas de servidor definen valores de configuración al crear servidores de middleware.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > Plantillas o seleccioneServidores, seleccione un servidor de middleware específico bajo Tipos de servidor y pulse Plantillas.

Botón Acción resultante
Nueva Abre un asistente que ayuda a crear una nueva plantilla de servidor de middleware. Los tipos de servidor para los que se pueden crear plantillas incluyen servidores de aplicaciones, servidores genéricos, direccionadores On Demand, servidores proxy o servidores Web. También se pueden crear o añadir plantillas de servidor para servidores de middleware que ya existan en su entorno como, por ejemplo, PHP, Apache Tomcat, BEA WebLogic Server, JBoss, servidor HTTP personalizado, Websphere Application Server externo, servidor de perfiles Liberty o servidor de WebSphere Application Server Community Edition.
Suprimir Elimina la plantilla de servidor.

Nombre

Especifica el nombre de la plantilla de servidor.

Plataforma

Especifica el tipo de plataforma en que está en ejecución el servidor.

Versión

Especifica qué versión de WebSphere Application Server Network Deployment está instalada y en ejecución en el servidor.

Descripción

Muestra la descripción de la plantilla de servidor.
