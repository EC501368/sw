Valores de plantilla de servidor de middleware

Esta página sirve para configurar los valores de una nueva plantilla de servidor que haya creado. Los tipos de servidor que existen para crear una plantilla incluyen a servidor de aplicaciones, genérico, direccionador On Demand, proxy o servidor web. También se pueden crear o añadir plantillas de servidor para un servidor de middleware que ya exista en su entorno como, por ejemplo, PHP, Apache Tomcat, BEA WebLogic Server, JBoss, servidor HTTP personalizado, servidor de perfiles Liberty, WebSphere® Application Server Community Edition o External WebSphere Application Server.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > Plantillas > Nueva o seleccione Servidores, seleccione un tipo de servidor middleware específico en Tipos de servidor y pulse Plantillas > Nueva.

Si desea editar una plantilla del servidor PHP, consulte la sección Configuración externa para obtener más información.

Paso 1: Seleccionar un servidor existente

Seleccione un servidor que ya exista a partir del cual cree una plantilla de servidor.

Habilite el recuadro de selección Seleccionar y pulse Aceptar. Si pulsa Servidores > Todos los servidores > Plantillas > Nueva, deberá seleccionar en primer lugar el tipo de servidor en la lista. En el caso de servidores de WebSphere Application Server Community Edition, se puede seleccionar de forma opcional la ubicación del entorno de tiempo de ejecución del servidor que se utiliza para crear el servidor.

Los tipos de servidor que hay disponibles para elegir son:
  • WebSphere Application Server
  • Servidor de perfiles Liberty
  • servidor proxy de WebSphere
  • Direccionador On Demand
  • Servidor PHP
  • Servidor WebSphere Application Server Community Edition
  • Servidor genérico
  • Servidor WebSphere MQ
  • Servidor web
  • Servidor Apache Tomcat
  • Servidor BEA WebLogic
  • Servidor JBoss
  • Servidor de aplicaciones de WebSphere externo
  • Servidor Apache
  • Servidor HTTP personalizado

Pulse Aceptar para continuar definiendo valores o pulse Cancelar.

Paso 2: Especificar propiedades específicas de la plantilla

Especifique el nombre y una descripción de la plantilla de servidor.

Pulse Aceptar después de escribir el nombre y una descripción para volver a la página principal que muestra el nombre, la plataforma, la versión y la descripción de la plantilla de servidor que acaba de crear.

Paso 3: Guardar o descartar los cambios

Seleccione Guardar para guardar los cambios directamente en la configuración maestra, o bien pulse Revisar para ver un resumen de los cambios antes de guardarlos o descartarlos.
