Valores de servidor Apache TomCat

Esta página sirve para ver configuraciones del servidor Apache TomCat.

Pulse Servidores > Todos los servidores > servidor_tomcat o bien pulse Servidores > Otros servidores de middleware > Servidores Apache TomCat > servidor_tomcat para ver esta página de la consola administrativa.

Nombre

Especifica un nombre para el servidor. El nombre para un servidor debe ser exclusivo en la célula.

El campo de nombre no puede contener los caracteres siguientes: # \ / , : ; " * ? < > | = + & % ' . El nombre definido debe ser exclusivo entre los servidores y no puede comenzar con un punto (.) ni con un espacio. Un espacio no genera un error, pero los espacios al principio o al final se eliminan automáticamente.

Nombre de nodo

Especifica el nombre de nodo asociado con el servidor. Normalmente, un nombre de nodo es el mismo que el nombre de host del sistema. Sin embargo, existe la posibilidad de elegirlo. Se puede hacer que el nombre de nodo sea distinto del nombre de host.

Conector HTTP

Especifica las propiedades específicas del servidor, incluidos el número de puerto y la habilitación.

Según sea el tipo de servidor, se pueden mostrar otros campos. Por ejemplo, los servidores PHP tienen campos para la ubicación raíz de Apache Tomcat y PHP. HTTP y HTTPS no se muestra con todos los tipos de servidor.

Conector HTTPS

Especifica las propiedades específicas del servidor, incluidos el número de puerto, la habilitación y la seguridad.
