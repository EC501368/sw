Configuración de clústeres de direccionadores On Demand

Utilice esta página para modificar la configuración de un clúster de direccionador On Demand (ODR). Un clúster ODR está formado por un grupo de direccionadores ODR que tienen una configuración común.

Para ver esta página de la consola administrativa, pulse Servidores > Clusters > Clústeres de direccionadores On Demand > nombre_clúster_ODR > Configuración.

Hay que poseer privilegios administrativos de configurador o administrador para cambiar los valores del clúster ODR.

Nombre de clúster

Indica un nombre lógico para el clúster ODR.

Se trata de un campo de sólo lectura.

Nombre del grupo de nodos de delimitación

Indica el nombre del grupo de nodos al que está asociado el clúster ODR. De forma predeterminada, se muestra el grupo de nodos DefaultNodeGroup.
