Crear un nuevo direccionador On Demand

Esta página sirve para crear un nuevo direccionador On Demand.

Para ver esta página de la consola administrativa, pulse Servidores > Todos los servidores > Nuevo. Seleccione Crear una nueva instancia de servidor y elija el tipo Direccionador On Demand. Puede ver también esta página de la consola administrativa, si pulsa Servidores > Tipos de servidores > Direccionadores On Demand > Nuevo.

Paso 1: Seleccionar un nodo

Seleccione un nodo.
Elija el nodo en que desee ejecutar el direccionador On Demand.
Nombre de servidor
Especifica un nombre para el direccionador On Demand que sea único en la célula.

El campo de nombre no puede contener los caracteres siguientes: # \ / , : ; " * ? < > | = + & % ' . El nombre definido debe ser exclusivo entre grupos servers y no puede comenzar por un punto (.) ni con un espacio. Un espacio no genera un error, pero los espacios al principio o al final se eliminan automáticamente.

Paso 2: Especificar propiedades específicas del servidor

Especificar propiedades específicas del servidor.

Protocolos soportados
Los protocolos soportados HTTP y SIP están seleccionados de forma predeterminada. Desmarque el recuadro si desea inhabilitarlo.
Generar puertos únicos
La opción para generar puertos únicos está seleccionada de forma predeterminada. Desmarque el recuadro si desea inhabilitarlo.

Paso 3: Seleccionar una plantilla de servidor

Elija de la lista la plantilla de direccionador On Demand que mejor especifique los atributos del direccionador On Demand que desea crear.

Se muestran el tipo y la descripción de la plantilla.

Paso 4: Confirmar nuevo servidor

Muestra un resumen de las acciones para crear el direccionador On Demand. Pulse Finalizar para crear el direccionador On Demand.
