  • Crear un nuevo servidor PHP
    Esta página sirve para crear servidores PHP (PHP Hypertext Preprocessor). Los servidores PHP están definidos como un servidor Apache con el módulo mod_php. La configuración de servidores PHP permite gestionar aplicaciones PHP en el mismo entorno administrativo que las aplicaciones Java EE (Java Platform, Enterprise Edition).
