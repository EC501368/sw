Creación de un nuevo servidor PHP

Esta página sirve para crear servidores PHP (PHP Hypertext Preprocessor). Los servidores PHP están definidos como un servidor Apache con el módulo mod_php. La configuración de servidores PHP permite gestionar aplicaciones PHP en el mismo entorno administrativo que las aplicaciones Java EE (Java Platform, Enterprise Edition).

Puede acceder a esta página de la consola administrativa de varias formas:
  • Pulse Servidores > Todos los servidores > Nuevo. Seleccione Crear una nueva instancia de servidor y elija el tipo Servidor PHP.
  • Pulse Servidores > Servidor nuevo. Seleccione Crear una nueva instancia de servidor y elija el tipo Servidor PHP.
  • Pulse Servidores > Tipos de servidores > Servidores PHP > Nuevo.

Paso 1: Seleccionar un nodo

Seleccione un nodo.
Elija el nodo en que desee ejecutar el servidor PHP.
Nombre de servidor
Especifica un nombre para el servidor PHP que es exclusivo en el nodo.

El campo de nombre no puede contener los caracteres siguientes: # \ / , : ; " * ? < > | = + & % ' . El nombre definido debe ser exclusivo entre grupos de servidores y no puede comenzar por un punto (.) ni por un espacio. Un espacio no genera un error, pero los espacios al principio o al final se eliminan automáticamente.

Paso 2: Seleccionar un entorno de ejecución Apache

Elija un entorno de tiempo ejecución Apache para su servidor PHP.

La página muestra los entornos de tiempo de ejecución Apache válidos en el nodo seleccionado, incluidas la versión y la vía de acceso de instalación.

Paso 3: Seleccionar un entorno de tiempo de ejecución PHP

Elija una versión de entorno de tiempo ejecución PHP para su servidor PHP.

La página muestra los entornos de tiempo de ejecución PHP válidos en el nodo seleccionado, incluidas la versión y la vía de acceso de instalación.

Paso 4: Seleccionar una plantilla PHP

Seleccione una plantilla de servidor para utilizar como modelo para el nuevo servidor PHP. Se puede seleccionar una plantilla, o correlacionarlo un servidor que ya exista.

  • Seleccione Crear el servidor PHP utilizando una plantilla de servidor PHP para crear el servidor PHP a partir de una plantilla de servidor PHP definida. Seleccione una plantilla de servidor de la lista.
  • Seleccione Crear el servidor PHP utilizando un servidor PHP existente para crear el servidor PHP después de un servidor PHP existente. Seleccione un servidor de la lista.

Paso 5: Confirmar nuevo servidor PHP

Muestra un resumen de las acciones para crear el servidor PHP. Pulse Finalizar para crear el servidor PHP.
