Normas HTTP

Vaya a los temas de ayuda de esta sección.

  • Normas HTTP de política de direccionamiento
    Esta página permite asociar normas HTTP de política de direccionamiento con direccionadores On Demand (ODR) y clústeres ODR.
  • Generador de expresiones de normas HTTP
    El generador de subexpresiones sirve para generar condiciones complejas de normas utilizando AND, OR y NOT así como la agrupación con paréntesis. El generador de normas valida la norma al aplicar los cambios y avisa si hay paréntesis desemparejados u operadores lógicos a los que no se les da soporte.
  • Generador de subexpresiones de acciones de normas de direccionamiento HTTP
    El generador de subexpresiones sirve para generar condiciones complejas de normas utilizando AND, OR y NOT así como la agrupación con paréntesis. El generador de subexpresiones valida la norma al aplicar los cambios y avisa si hay paréntesis desemparejados u operadores lógicos a los que no se les da soporte.
