Valores del servicio de datos de visualización

Esta página sirve para configurar los datos históricos de registro de anotaciones cronológicas, con el propósito de crear informes sobre datos históricos, para calcular cuentas de cargo o para planificar la capacidad.

Para ver esta página de la consola administrativa, pulse Administración del sistema > Servicio de datos de visualización.

Si está utilizando un rol de operador o supervisor, sólo puede ver la información del servicio de datos de visualización. Si se posee un rol de configurador, se puede cambiar la configuración. Si posee un rol de administrador, posee todos los privilegios para el servicio de datos de visualización.

Habilitar registro

Seleccione esta opción para iniciar el registro de anotación cronológica de datos históricos.

Formato de la indicación de la hora

Específica el patrón de fecha y hora que se utilizará al registrar datos de visualización.

Utilice la clase Java SimpleDateFormat para dar formato su indicación de fecha y hora. Por ejemplo, para la salida de indicación de fecha y hora 12.06.2006 5:26:30:978 PM PDT, utilice el siguiente valor de formato de indicación de fecha y hora: 

MM.dd.yyyy hh:mm:ss:SSS aaa z

Consulte la documentación de la API para obtener más información sobre la clase Java SimpleDateFormat.

Si está utilizando IBM Tivoli Usage and Accounting Manager, utilice un formato que separa la fecha y la hora en campos distintos.

Tamaño máximo de archivo

Escriba un número entero para indicar el tamaño máximo de archivo de registro.

Número máximo de archivos históricos

Especifica un valor entero que indica el número máximo de archivos de anotaciones cronológicas que se pueden generar.

Nombre de archivo

Especifica la vía de acceso en la que se generarán los archivos de registro.

Intervalo de escritura de las anotaciones de datos

Especifica el tiempo entre la escritura de dato en los archivos de anotaciones cronológicas en segundos, minutos, horas o días. El intervalo de escritura en las anotaciones cronológicas de datos es un número entero entre 1 y 2147483647.

Acción del transformador de datos

Especifica la forma en que manejar los puntos adicionales que se reciben dentro del intervalo de escritura de anotaciones cronológicas de datos.

Por ejemplo, si el intervalo de escritura es de un minuto, y los datos se reciben cada 15 segundos, quiere decir que en el intervalo se reciben cuatro puntos de datos. Para descartar los tres primeros puntos de datos y escribir únicamente el punto de datos en el intervalo de un minuto en las anotaciones cronológicas, seleccione Omitir. Para obtener la media de los cuatro puntos de datos al final del intervalo y escribir la media en las anotaciones cronológicas, seleccione Promedio.
