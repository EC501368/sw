Políticas de direccionamiento de aplicaciones

Utilice esta página para definir las políticas de direccionamiento de las aplicaciones OSGi. Las políticas de direccionamiento le permiten direccionar peticiones HTTP a la edición de la aplicación adecuada.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Tipos de aplicación > Aplicaciones de nivel empresarial > nombre_aplicación > nombre_activo > Políticas de direccionamiento.

Clases de trabajo para peticiones HTTP

Expanda esta opción para ver o gestionar las clases de trabajo para solicitudes HTTP.

Nueva

Crea una clase de trabajo.

El asistente permite crear una clase de trabajo y especificar propiedades generales.

Suprimir

Suprime una clase de trabajo.

Editar patrones HTTP

Edita los patrones HTTP que se utilizan para hacer coincidir una petición HTTP con una regla existente.

Añadir regla

Crea una regla nueva.

Suprimir regla

Suprime una regla existente.

Subir, bajar

Da prioridad a las normas en el caso de varias normas. El orden de las normas es clave para determinar las coincidencias. Si se produce una coincidencia en la primera norma, las normas subsiguientes no se ejecutan.

Si la solicitud HTTP coincide con

Establece la condición de coincidencia HTTP de una regla que ya existe.

Seleccionar acción

Asigna la acción de direccionamiento por omisión a una de las opciones siguientes.
Permitir direccionamiento a
El nombre de edición se muestra automáticamente para la opción seleccionada.
Permitir direccionamiento con afinidad a
El nombre de edición se muestra automáticamente para la opción seleccionada.
Redirigir el direccionamiento a
En el campo Entre el URI al que redirigir, escriba el URI.
Rechazar direccionamiento con código de retorno
En el campo Entre el código de retorno, escriba el código de retorno HTTP.
Permitir direccionamiento a servidores en modalidad de mantenimiento
El nombre de edición se muestra automáticamente para la opción seleccionada.
Permitir direccionamiento con afinidad a servidores en modalidad de mantenimiento
El nombre de edición se muestra automáticamente para la opción seleccionada.

Aplicar

Confirma los nuevos valores de la norma.

Pulse Aplicar para confirmar el valor de regla.
