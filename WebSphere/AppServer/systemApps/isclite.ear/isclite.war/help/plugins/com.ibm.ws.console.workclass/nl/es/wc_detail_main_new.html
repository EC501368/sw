Definir propiedades generales de las clases de trabajo

Utilice esta página para crear clases de trabajo. Para ver esta página, pulse Aplicaciones > Aplicaciones de empresa > Seleccione una aplicación > Políticas de servicio y pulse Nuevo para el tipo de petición.

Se proporciona información ampliada sobre los campos de la política de clases de trabajo.

Nombre El campo de nombre es necesario y es un campo definido por el usuario. El campo de nombre no puede contener los caracteres siguientes: ]]> \ / , : ; " * ? < > | = + & % ' # El nombre debe ser exclusivo entre las políticas de salud de una célula y no puede empezar por un punto (.) o un espacio. Un espacio no genera un error, pero los espacios al principio o al final se eliminan automáticamente. Se recomienda utilizar nombres de clase de trabajo coherentes y con sentido. Por ejemplo, una clase de trabajo de intercambio de valores puede llamarse TrabajoIntercambioValores.
