Confirmar creación de clase de trabajo

Utilice esta página para confirmar las propiedades de la nueva clase de trabajo. Para ver esta página, pulse Aplicaciones > Aplicaciones de empresa > Seleccione una aplicación > Políticas de servicio y como tipo de petición, pulse Nuevo. Esta es la última página del asistente de creación de clases de trabajo.

En esta última página del asistente de creación de clases de trabajo, puede revisar la clase de trabajo y pulsar Anterior para revisar las opciones o pulsar Finalizar para confirmarlas y crear la clase de trabajo.
