Gestores autónomos

Vaya a los temas de ayuda de esta sección.

  • Añadir y eliminar valor
    Utilice esta página para definir acciones para añadir y eliminar operaciones de elasticidad. Asegúrese de que se han creado acciones personalizadas antes de definir las acciones para añadir y eliminar operaciones de elasticidad.
  • Propiedades personalizadas del controlador de ubicación de aplicaciones
    Utilice esta página para visualizar y gestionar una lista de parejas de datos arbitrarias de nombre-valor, donde el nombre es una clave de propiedad y el valor es un valor de serie de texto que se puede utilizar para establecer las propiedades internas de configuración del sistema. El definir una nueva propiedad permite configurar un valor más allá de lo que está disponible en la consola administrativa.
  • Valores del controlador de ubicación de aplicaciones
    Utilice esta página para definir los valores que controlan la ubicación de las aplicaciones. Se puede habilitar el controlador de ubicación autónomo, definir con qué frecuencia se darán cambios de ubicación y controlar el comportamiento del controlador de ubicación cuando opere en la modalidad supervisada.
  • Valores del gestor de flujo de peticiones autónomo
    Utilice este panel para configurar el gestor de flujo de solicitudes autónomo (ARFM). El gestor de flujo de solicitudes autónomo (ARFM) gestiona los mensajes entrantes para las aplicaciones y determina si los servidores de middleware pueden darles servicio y cuándo pueden hacerlo. Estas acciones de gestión tienen lugar en direccionadores On Demand para el tráfico SIP (Session Initiation Protocol) y HTTP y en los servidores de aplicaciones para los mensajes JMS (Java Message Service) e IIOP (Inter-ORB Protocol).
  • Colección de acciones personalizadas de elasticidad
    Utilice esta página para gestionar acciones definidas por el usuario que se pueden realizar con operaciones de elasticidad. Las operaciones de elasticidad permiten añadir o eliminar dinámicamente nodos de la célula para aumentar o disminuir los recursos del sistema.
  • Valores de acciones personalizadas de elasticidad, sin utilizar Java
    Utilice esta página para gestionar acciones definidas por el usuario que no utilicen Java y que se puedan realizar con operaciones de elasticidad. Las operaciones de elasticidad permiten añadir o eliminar dinámicamente nodos de la célula para aumentar o disminuir los recursos del sistema.
  • Valores de acciones personalizadas de elasticidad, utilizando Java
    Utilice esta página para modificar acciones Java definidas por el usuario que se pueden realizar con operaciones de elasticidad. Las operaciones de elasticidad permiten añadir o eliminar dinámicamente nodos de la célula para aumentar o disminuir los recursos del sistema.
  • Colección de operaciones de elasticidad
    Utilice esta página para seleccionar acciones para añadir y eliminar operaciones de elasticidad. Las operaciones de elasticidad permiten añadir o eliminar dinámicamente nodos de la célula para aumentar o disminuir los recursos del sistema.
