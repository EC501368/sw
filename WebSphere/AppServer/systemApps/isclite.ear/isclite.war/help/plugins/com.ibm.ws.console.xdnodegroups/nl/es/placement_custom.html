Propiedades personalizadas del controlador de ubicación de aplicaciones

Utilice esta página para visualizar y gestionar una lista de parejas de datos arbitrarias de nombre-valor, donde el nombre es una clave de propiedad y el valor es un valor de serie de texto que se puede utilizar para establecer las propiedades internas de configuración del sistema. El definir una nueva propiedad permite configurar un valor más allá de lo que está disponible en la consola administrativa.

Para ver esta página de la consola administrativa, pulse Políticas operativas > Gestores autónomos > Controlador de ubicación de aplicaciones > Propiedades personalizadas > nuevo.

Nombre

Especifica el nombre, o clave, de la propiedad.

Cada propiedad debe ser única. Si se utiliza el mismo nombre para varias propiedades, se utiliza el valor especificado para la primera propiedad con dicho nombre. No comience los nombres de propiedades por was. Este prefijo está reservado para las propiedades predefinidas en WebSphere Application Server.

Valor

Especifica el valor que está emparejado con el nombre especificado.

El valor es una serie de texto que permite establecer propiedades internas de configuración del sistema.

Descripción

Proporciona información descriptiva acerca de la pareja nombre-valor.

Pulse Aplicar o Aceptar para guardar los valores.
