Valores de filtros de la Consola administrativa

Utilice los valores de filtro para especificar cómo filtrar las entradas que se muestran en una vista de colección.

Seleccione la columna que debe filtrarse y, luego, especifique los criterios de filtro.

Columna a filtrar
Utilice la lista desplegable para seleccionar la columna que debe filtrarse. Al aplicar el filtro, la tabla de colecciones se modifica consiguientemente.

Por ejemplo, seleccione Servidores de aplicaciones si desea especificar criterios según los que se filtrarán los nombres de los servidores de aplicaciones.

Criterios de filtro
En el campo que aparece junto a la lista desplegable, especifique una serie que deba encontrarse en el nombre de una entrada de colección para que dicha entrada pueda aparecer en la tabla de colecciones. La serie puede contener los caracteres %, * o ? como caracteres comodín. Por ejemplo, escriba *Ap* para buscar cualquier servidor de aplicaciones cuyo nombre contenga la serie Ap.

Utilice el carácter \ delante de cada uno de los caracteres ( ) ^ * % { } \ + $ de modo que el motor de expresiones regulares que realice la búsqueda establezca coincidencias correctas según los criterios de búsqueda especificados. Por ejemplo, para buscar todos los proveedores JDBC que contienen (XA) en el nombre del proveedor, especifique la serie siguiente: 

*\(XA\)
