Añadir y eliminar valor

Utilice esta página para definir acciones para añadir y eliminar operaciones de elasticidad. Asegúrese de que se han creado acciones personalizadas antes de definir las acciones para añadir y eliminar operaciones de elasticidad.

Para ver esta página de la consola administrativa para la operación de añadir, pulse Políticas operativas > Gestores autónomos > Controlador de ubicación de aplicaciones > Operación de elasticidad > Añadir.

Para ver esta página de la consola administrativa para la operación de eliminar, pulse Políticas operativas > Gestores autónomos > Controlador de ubicación de aplicaciones > Operación de elasticidad > Eliminar.

Nombre

Especifica el nombre de la operación de elasticidad.

Descripción

Opcionalmente, especifica la descripción de la operación de elasticidad.

Operación de elasticidad

Especifica el tipo de operación de elasticidad. El valor es Añadir para la operación de añadir y Eliminar para la operación de eliminar.

Acción de añadir, acción de eliminar

Especifica las acciones para la operación de elasticidad que puede añadir o eliminar de la tabla.

Subir, bajar

Prioriza las acciones en el orden en que se ejecutan en el caso de varias acciones.

Paso

Especifica un valor entero que se asigna automáticamente a la acción, según el orden en que la acción aparece en la tabla.

Acción

Especifica el nombre de la acción que se añade cuando pulsa Acción Añadir.

Aplicar

Confirma los nuevos valores de añadir.

Pulse Aplicar para confirmar los cambios.
