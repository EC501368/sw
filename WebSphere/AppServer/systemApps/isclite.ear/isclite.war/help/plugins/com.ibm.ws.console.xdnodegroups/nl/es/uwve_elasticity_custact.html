Colección de acciones personalizadas de elasticidad

Utilice esta página para gestionar acciones definidas por el usuario que se pueden realizar con operaciones de elasticidad. Las operaciones de elasticidad permiten añadir o eliminar dinámicamente nodos de la célula para aumentar o disminuir los recursos del sistema.

Para ver esta página de la consola administrativa, pulse Políticas operativas > Gestores autónomos > Controlador de ubicación de aplicaciones > Acciones personalizadas de elasticidad.

Nombre

Especifica el nombre de la acción personalizada de elasticidad.

SO soportados

Especifica los sistemas operativos a los que se aplica la acción.

Acción

Especifica el ejecutable para la acción.
