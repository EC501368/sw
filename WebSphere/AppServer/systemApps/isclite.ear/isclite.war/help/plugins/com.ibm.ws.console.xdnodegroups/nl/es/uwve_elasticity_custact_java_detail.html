Valores de acciones personalizadas de elasticidad, utilizando Java

Utilice esta página para modificar acciones Java definidas por el usuario que se pueden realizar con operaciones de elasticidad. Las operaciones de elasticidad permiten añadir o eliminar dinámicamente nodos de la célula para aumentar o disminuir los recursos del sistema.

Para ver esta página de la consola administrativa, pulse Políticas operativas > Gestores autónomos > Controlador de ubicación de aplicaciones > Acciones personalizadas de elasticidad > nombre_acción.

Nombre

Especifica el nombre de la acción personalizada de elasticidad.

Ubicación del directorio ejecutable de Java

Especifica la ubicación del directorio ejecutable Java en el servidor.

Tipo de destino ejecutable

Especifica si el programa ejecutable es una clase Java o un archivo jar.

Ejecutable

Especifica el ejecutable del programa que realiza la acción personalizada de elasticidad.

Argumentos ejecutables

Especifica los argumentos que el ejecutable del programa debe ejecutar.

Nombre de una variable para hacer referencia a un nombre de usuario en argumentos utilizados en la ejecución

Especifica el nombre de una variable que puede utilizar para hacer referencia a un nombre de usuario en los argumentos para el programa ejecutable.

Nombre de usuario a sustituir para la variable de nombre de usuario en el momento de la invocación

Especifica el nombre de usuario que puede sustituir para la variable que se utiliza para hacer referencia a un nombre de usuario.

Nombre de una variable para hacer referencia a una contraseña en argumentos utilizados en la ejecución

Especifica el nombre de una variable que puede utilizar para hacer referencia a una contraseña en los argumentos para el programa ejecutable.

Contraseña a sustituir para la variable de contraseña en el momento de la invocación

Especifica la contraseña que puede sustituir por la variable que se utiliza para hacer referencia a una contraseña.

SO soportados

Especifica los sistemas operativos a los que se aplica la acción.

Nombre de una variable de entorno que almacena un nombre de archivo PID

Opcionalmente especifica el nombre de la variable de entorno que contiene la vía totalmente calificada para un archivo. El archivo contiene el ID de proceso para la operación que se está ejecutando. Si se especifica ninguna variable de entorno, la operación se ejecutará en primer plano.

Aplicar

Confirma los nuevos valores.

Pulse Aplicar para confirmar los cambios.
