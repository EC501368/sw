Informes de aplicaciones

Los diagramas personalizados permiten mostrar si se están cumpliendo los objetivos. Para darse cuenta de los beneficios de utilizar este entorno virtualizado, es importante saber el rendimiento que presentan las aplicaciones. La creación de diagramas dinámicos proporciona una perspectiva visual de este rendimiento. De forma específica, hay disponibles estadísticas como la disponibilidad, el tiempo de respuesta, el tráfico y el rendimiento. Se proporciona una amplia gama de opciones a partir de las cuales se pueden crear distintos diagramas.

Para ver esta página de la consola administrativa, pulse la pestaña Aplicaciones > Todas las aplicaciones > nombre_aplicación > Operaciones.

Preferencias de informe

Especifica las preferencias globales para todos los grupos de diagramas.

Grupo de diagramas por omisión
Especifica el grupo de diagramas que por omisión se muestra en esta página.
Tipo de diagrama predeterminado
Especifica el formato de la imagen del diagrama que se muestra. Se puede elegir entre SVG (Scalable Vector Graphics) o JPEG. Se necesita SVG para disponer la funcionalidad completa del diagrama. El navegador debe tener un plug-in SVG, o una implementación instalada, para ver diagramas en SVG.
Tipo de diagrama por omisión
Especifica el tipo de diagrama predeterminado al crear uno nuevo: línea, área, barra vertical, barra apilada vertical 3-D, barra horizontal, barra apilada horizontal 3-D. Cuando se cambia este valor, no se aplica a ningún otro diagrama que ya se haya creado. Pulse Preferencias en el diagrama para cambiar uno que ya exista.
Tamaño de diagrama por omisión
Especifica el tamaño de diagrama predeterminado cuando se crea un nuevo diagrama: pequeño, medio o grande. Cuando se cambia este valor, no se aplica a ningún otro diagrama que ya se haya creado. Pulse Preferencias en el diagrama para cambiar uno que ya exista.
Mostrar objetivos de la política
Al crear un nuevo diagrama, si se habilita, se muestra el valor de objetivo que se ha establecido para la métrica de datos, que se puede comparar con los valores actuales. Cuando se cambia este valor, no se aplica a ningún otro diagrama que ya se haya creado. Pulse Preferencias en el diagrama para cambiar uno que ya exista.
Mostrar tipos de conjuntos de datos
Si se habilita, se muestran los tipos de conjunto de datos para cada punto de datos. Cuando se cambia este valor, no se aplica a ningún otro diagrama que ya se haya creado. Pulse Preferencias en el diagrama para cambiar uno que ya exista.
Habilitar renovación automática
Si se habilita, los diagramas se renuevan de forma automática en cada intervalo de renovación de diagrama que se haya especificado.
Intervalo de renovación de diagramas
Especifica un número de segundos entre cada renovación de diagrama. Si se tiene un gran número de datos en los diagramas, considere establecer este valor con un intervalo mayor.

Botones de creación de diagrama

Para crear un nuevo diagrama dentro del grupo de diagramas, pulse Abrir una nueva pestaña de diagrama. Para abrir el diagrama seleccionado actualmente en una nueva ventana, pulse Ver diagrama en nueva ventana. Para devolver el diagrama desde la nueva ventana al grupo de diagramas, pulse Ver diagrama en grupo de diagramas.

Preferencias de diagrama

Pulse Preferencias para actualizar las preferencias del diagrama concreto. Estos cambios únicamente se aplican al diagrama seleccionado actualmente.

Tipo de diagrama
Cambia el aspecto del diagrama. Se pueden crear varios tipos distintos de diagramas de líneas y de barras:
  • Línea
  • Área
  • Barra vertical
  • Barra apilada vertical 3-D
  • Barra horizontal
  • Barra apilada horizontal 3-D
Tamaño de diagrama
Modifica el tamaño del diagrama. Se puede elegir pequeño, mediano o grande.
Mostrar tipos de conjuntos de datos
Especifica si se muestran los tipos de conjuntos de datos que se muestran en la columna Patrón de la columna de la tabla de datos de diagrama.
Mostrar objetivos de la política
Especifica si los datos se muestran junto con los objetivos que se han configurado.

Opciones de datos de diagrama

Estas opciones permiten cambiar los datos que se muestran en el diagrama.

Añadir datos...
Añade conjuntos de datos al diagrama. Se pueden configurar varios conjuntos de datos para mostrarlos en el mismo diagrama.
Eliminar
Elimina los datos seleccionados del diagrama. Seleccione los datos que desee eliminar de la tabla y pulse Eliminar.
Ver tabla
Muestra una tabla de los datos en el diagrama.

Tabla de diagrama

La tabla de diagrama muestra una lista de datos que se incluyen en el diagrama.

Patrón
Especifica la forma que se visualiza en el diagrama, si se está utilizando un diagrama de líneas. No se puede cambiar el patrón.
Tipo de conjunto de datos
Especifica el tipo del conjunto de datos con los que está realizando un seguimiento de las métricas.
Conjunto de datos
Indica el objeto que está proporcionando los datos en tiempo de ejecución. Pulse Añadir datos... para añadir nuevos objetos para realizar su seguimiento.
Métrica de los datos
Visualiza las métricas seleccionadas para realizar el seguimiento para este conjunto de datos.
Filtro de datos
Con la ayuda del campo de Filtro de datos se pueden seleccionar proxies existentes o direccionadores On Demand y filtrar la correlación consecuentemente. Esta selección resulta útil para la resolución de problemas del entorno.
Escala
Indica el tamaño de los datos representados.

Opciones de grupo de diagramas

Para guardar el grupo de diagramas, entre un nombre y pulse Guardar.

Para eliminar el actual grupo de diagramas, pulse Eliminar grupo de diagramas.
