Tareas activas de política de servicio

Esta página sirve para ver todas las tareas que están activas de una política de servicio específica.

Pulse Políticas operativas > Políticas de servicio > seleccione una política de servicio > pestaña Operaciones > pestaña Tareas activas para ver esta página administrativa.

Acción

Especifica la acción que se puede realizar para una política de servicio específica.

Seleccione el recuadro de selección junto a la tarea en la que desea realizar una acción y, a continuación, seleccione la acción del menú desplegable.

ID de tarea

Especifica el ID en forma de un enlace al objeto de destino.

Estado

Especifica el estado de la política de servicio. Por ejemplo, nueva, en progreso o caducada.

Gravedad

Muestra la gravedad de la política de servicio.

Hora de inicio

Especifica la hora en que se originó la política de servicio.
