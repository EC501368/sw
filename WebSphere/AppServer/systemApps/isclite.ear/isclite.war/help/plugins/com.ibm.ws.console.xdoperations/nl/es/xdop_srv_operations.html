Pestaña operaciones de servidor

Esta página sirve para ver el estado global del entorno del servidor.

Para ver esta página administrativa, pulse la pestaña Servidores > Todos los servidores > nombre_servidor > Operaciones.

Esta vista muestra lo que está ocurriendo actualmente en el entorno del servidor. Se incluye información sobre el estado global del servidor, la gestión de solicitudes, la información de asignación y la información de tareas.

Nombre de servidor

Especifica el nombre del servidor.

Nodo

Especifica el nodo en el que está instalado el servidor.

Tipo

Especifica el tipo de servidor.

Nombre de clúster

Especifica el nombre del clúster al que pertenece el servidor.

Estado

Indica si el servidor está iniciado, detenido o si no está disponible.

Tabla 1. Estado del servidor
Icono Estado Descripción
Se está ejecutando parcialmente El servidor está en el proceso de cambiar de un estado de Detenido a un estado de Iniciado. Se está comenzando a ejecutar el servidor pero todavía no lo está haciendo en su totalidad.
Iniciado Se ha iniciado el servidor.
Deteniendo El servidor está en el proceso de cambiar de un estado de Iniciado a un estado de Detenido.
Detenido El servidor no se está ejecutando.
Desconocida No se puede determinar el estado del servidor.

Estabilidad

Especifica el estado operativo del servidor.

Tabla 2. Estabilidad del servidor
Icono Estado Descripción
Estable El servidor está operativo y sin problemas.
Cuestionable El servidor tiene algunos problemas, pero sigue operativo.
Inestable El servidor tiene graves problemas.
Desconocida No se puede informar sobre la estabilidad del servidor.

Sesiones reales

Especifique el número de sesiones locales que están almacenadas actualmente en la memoria caché. Se puede controlar la estadística, por ejemplo, cuando se pone el servidor en la modalidad de mantenimiento para completar la operación de inmovilización.

Peso

Especifica el peso de direccionamiento asignado a los destinos de despliegue del servidor.

Gestión de solicitudes

Muestra los direccionadores On Demand (ODR) que actualmente están direccionando trabajo al servidor, las aplicaciones en el servidor que están recibiendo trabajo, así como el trabajo que se ha clasificado para las políticas de servicio desde que se inició el ODR.

Asignación

Muestra los otros procesos que están en ejecución en el mismo nodo que el servidor.

Tareas activas de servidor

Especifica las tareas pendientes del servidor.
